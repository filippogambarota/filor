# write_bib_rmd

write_bib_rmd

## Usage

``` r
write_bib_rmd(
  file = knitr::current_input(),
  input_bib,
  output_bib,
  get_file = FALSE
)
```

## Arguments

- file:

  character that indicate the input file

- input_bib:

  character that indicate the input `.bib` file

- output_bib:

  character that indicate the output `.bib` file

- get_file:

  logical should the raw file be invisibly returned

## Value

the bibfile
