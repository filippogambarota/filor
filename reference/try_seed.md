# try_seed

Repeat an R expression a certain number of times with a random seed.
Each iteration is self-paced by pressing a key. The random seed is
printed along with the result of the

## Usage

``` r
try_seed(expr, maxrun = 100, digits = 4)
```

## Arguments

- expr:

  an R expression

- maxrun:

  the maximum number of iterations

- digits:

  the number of digits for the random seed
